# 功能介绍
AI+爬虫技术，实现一句话就能自动监控页面内容，实时订阅你关心的内容

# 使用方法
1、输入你要监控的网页地址

2、一句话描述你想监控什么数据，比如：“有机器人相关的新闻通知我”、“有图书馆招聘公共告诉我”、“xxx商品上架了通知我”

3、AI自动分析（1-2分钟），实时监控，支持邮件、微信、短信、webhook等通知方式
